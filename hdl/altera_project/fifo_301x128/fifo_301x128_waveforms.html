Sample behavioral waveforms for design file fifo_301x128.v

The following waveforms show the behavior of dcfifo megafunction for the chosen set of parameters in design fifo_301x128.v. The design fifo_301x128.v has a depth of 128 words of 301 bits each. The fifo is in legacy synchronous mode. The data becomes available after 'rdreq' is asserted; 'rdreq' acts as a read request.
